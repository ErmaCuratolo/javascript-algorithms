# 图

在计算机科学中，**图**是一种抽象的数据结构，实现了来自数学的有向图和无向图的概念，更具体地说，是图论领域的概念。


图的抽象数据结构包括一组有限的、可能是可变的顶点或节点或点，以及这些元素的有序或无序对的集合。这些对是边、非定向弧或线段，对于非定向图，是线段，对于定向图，是箭头、定向边、弧或线段。顶点可以是结构的一部分，也可以是外部实体，由整数或引用表示。
